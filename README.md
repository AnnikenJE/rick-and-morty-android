## 👽 Case: Rick and Morty

The app allows users to browse characters from the Rick and Morty API, create and store their own characters locally, and interact with Rick and Morty related data across multiple screens.

The application focuses on functionality, clean and scalable code, API integration, local storage, navigation between screens, and a usable UI with proper user feedback and error handling.
